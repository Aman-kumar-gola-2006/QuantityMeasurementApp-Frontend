

# 📦 Quantity Measurement Frontend

This is an **Angular-based frontend application** for the Quantity Measurement system.
It allows users to **authenticate** and perform various **measurement operations** efficiently.

---

## 🚀 Features

### 🔐 Authentication

* User Login
* User Signup
* Secure communication using **JWT (JSON Web Token)**

---

### 📏 Measurement Types

* Length
* Weight
* Temperature
* Volume

---

### ⚙️ Operations

* Compare values
* Convert units
* Arithmetic operations:

  * Add
  * Subtract
  * Divide

---

### 📊 Additional Functionality

* History tracking of all operations

---

## 🛠️ Tech Stack

* **Angular**
* **TypeScript**
* **HTML5**
* **CSS3**

---

```
src/app/
```
